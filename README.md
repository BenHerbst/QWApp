# QWApp - WhatsApp Web on Linux

## 🧲 About
QWApp wraps the WhatsApp Web Application into a Linux desktop app. It supports features like a system tray and notifications. Built using Qt6 and Qt6-WebEngine, the main goal is stability and robust usage for every day.

## 🦸 Features 
- Native Notifications
- System tray icon ( App runs in the background for notifications and fast start )
- Auto-startup on system boot
- Menubar

## 🎁 Packaging
The project is packaged for Flatpak, Debian and Fedora to target many users.

You can find the .deb and .rpm files in the releases tab on GitHub.

## 👩‍💻 Development
QWApp uses C++ and Qt6 together with the Qt6-Webengine to render WhatsApp Web.

Built with CMake and CPack, DBus for native notifications.
